# **Neural Topological SLAM for Visual Navigation Review**

## **Abstract**

우리는 이미지 목표 내비게이션을 만들겠다. 노드와 Coarse 기하학적 연결과 구동 노이즈를 극복하겠다.

## Introduction

실제 우리 인간이 주어진 이미지를 보았을 때 찾는 방식은 미터 방식이 아님
하지만, 기존의 연구들은 미터법 단위로 정밀하게 표현하는 방식을 채택하곤함.
하지만,

1. **확장성의 한계(Scalability)**: 환경의 크기가 커지거나 로봇이 쌓는 경험의 양이 많아질수록 미터법 지도는 연산과 메모리 측면에서 확장하기가 어려움.
2. **현실적인 구동 노이즈(Actuation Noise)**: 실제 로봇이 움질일 때 이러한 미터법 단위의 지도를 일관적으로 그리는 것이 정말 까다롭고, 정확한 위치를 매번 추정할 수 있는 것도 아님.

[!NOTE]
이런 현실적인 문제를 해결하고자 한 것. 미터법 대신 위상학적 표현을 제안함.



## Related Work - 선제시..?

기본적으로 내비게이션 태스크는 두가지 카테고리가 존재함.

1. `목표위치를 알고 있어` 제한적인 탐색만 필요한 태스크
  -> 전자의 태스크는 효율적인 검색과 안정적인 제어 명령 실행을 목표로 함.
2. `목표위치를 모르는 상태`에서 탐색이 필수적인 태스크
  -> 후자의 태스크는 새로운 환경에서 효율적인 탐색을 위하여 의미론적 추론과 상식적 추론을 필요로함.

어쨌거나, 새로운 환경에서 **메모리 기반 정책**이 연구되어 왔음. -> LSTM // Transformer 인공신경망 메모리 형태
내비게이션을 위한 표현력이 풍부한 신경망 메모리 -> **공간 메모리, 위상학적** 접근 방식. -> 기존 방식은 문제가 있음.

1. 공간 메모리는, 미터법의 일관성 및 이산 상태의 공간표현, 실제 환경은 Discrete한 환경이 아니기에 잘 안먹힘.
2. 기존의 위상학적 접근 방식 -> 진정한 탐색 어려움.

-> 그니까 우리가 진정한 탐색을 하려고 함.

세가지 학습 방법론..

1. 강화학습 -> 정책 학습이 어렵고 샘플 효율성이 떨어져 포기
2. 모방학습 -> 샘플 효율성은 좋으나, 탐색적 행동을 배우기 어려움.
3. 자가 지도 학습 -> 기존 방식은 좀 제한적임 // 우리가 확장해보겠음.



## **사용하는 방법론(Method)**



### **Model component**

1. **그래프 업데이트 모듈**: 에이전트의 실시간 관측값을 기반으로 위상 지도(그래프)를 최신 상태로 업데이트합니다.
2. **글로벌 정책**: 그래프의 노드 중 하나를 **'장기 목표(Long-term Goal)'**로 선택하고, 경로 계획(Path Planning)을 이용해 그곳에 도달하기 위한 중간 목적지인 **서브골(Subgoal)**을 찾아냅니다.
3. 로컬 정책: 카메라로 보는 시각적 관측값에 의존하여 설정된 서브골까지 실제로 로봇을 제어하여 이동(Navigation)시킵니다.
